# CourierPro Dashboard

A modern courier/shipment tracking dashboard built with Bootstrap, Font Awesome, and jQuery.

## Features
- 📊 Dashboard with shipment overview
- ✨ Create new shipments
- 🔍 Track shipments by tracking number
- 👤 User profile section
- 📱 Responsive design (mobile-friendly)

## Technologies Used
- HTML5
- CSS3 (with custom variables)
- Bootstrap 5.3.3
- jQuery 3.6.0
- Font Awesome 6.5.0

## How to Use
1. Open `index.html` in a web browser
2. View all shipments on the dashboard
3. Click "Create Shipment" to add new shipments
4. Use "Track Shipment" to search by tracking number

## Sample Shipments
The dashboard comes with 3 sample shipments:
- CP1699000000 - Created (Alice → Bob)
- CP1699000001 - In Transit (Charlie → David)
- CP1699000002 - Delivered (Eve → Frank)
