# WordPress Core Files

This folder contains only the WordPress core files from `/var/www/html`.

## Files included
- `index.php` - Main WordPress entry point
- `wp-*.php` - WordPress core files
- `xmlrpc.php` - XML-RPC API
- `license.txt` - GPL License
- `readme.html` - WordPress readme

## Files excluded
- `wp-config.php` - Contains sensitive database credentials
- `wp-content/` - Themes, plugins, and uploads
- `wp-admin/` - WordPress admin files
- `wp-includes/` - WordPress core includes

## Note
This is just the root WordPress files. The actual site folders (hazara, Kirjasto, etc.) are stored separately.
